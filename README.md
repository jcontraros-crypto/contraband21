# Family Planner Meteor App

A simple shared Meteor app with three pages:

- Calendar
- Grocery List
- Wishlist

## Requirements

- Meteor installed

## Run locally

```bash
meteor npm install
meteor run
```

Open http://localhost:3000

## Notes

- No user accounts are included.
- Data is shared in the local app database.
- The calendar supports add, edit, and delete for appointments.
- Grocery List and Wishlist support add, edit, and delete.
